## üéØ By Organization Size

### [Startup Edition (1-10 people)](./guides/startup-edition.md)
Lightweight, practical governance that doesn't slow product delivery.
- Essential security controls only
- Informal policies (guidelines)
- 1-2 person governance effort
- Focus: Data protection & access control

### [SaaS Edition (10-50 people)](./guides/saas-edition.md)
Structured governance for growing companies.
- Formal policies & procedures
- SOC 2 Type II compliance
- Governance council
- 2-3 person governance team

### [Enterprise Edition (50+ people)](./guides/enterprise-edition.md)
Enterprise-grade governance with distributed accountability.
- Complete policy framework
- Multiple compliance standards
- Executive oversight committee
- 8-12 person governance team

## üìä Framework Domains

The framework is organized around **5 COBIT Domains**:

| Domain | Focus | Key Controls |
|--------|-------|-------------|
| **EDM** | Evaluate, Direct, Monitor | Strategic governance, risk oversight |
| **APO** | Align, Plan, Organize | Strategy alignment, architecture |
| **BAI** | Build, Acquire, Implement | System delivery, AI/API governance |
| **DSS** | Deliver, Service, Support | Operations, security, incident response |
| **MEA** | Monitor, Evaluate, Assess | Performance measurement, compliance |
